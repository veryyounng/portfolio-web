자기소개

저는 새로운 정보와 경험에 호기심을 가지고 주도적으로 배우는 태도를 가지고 있습니다.

이러한 호기심은 개발을 배우는 과정에서도 큰 원동력이 되었습니다. Spring Boot와 React를 활용해 REST API를 구현하고 Python 기반의 데이터 파이프라인도 직접 구축했습니다.

특히 팀의 협업 환경을 개선하고자 회사 내부 GitLab 서버를 직접 구축하고 CI/CD 자동 배포 환경을 구축하였습니다. 또한 Docker로 애플리케이션을 컨테이너화하고 AWS EC2에 배포하면서 클라우드 인프라 전반에 대한 실전 감각을 키웠습니다.

현재는 Kafka와 Redis를 학습하며 실시간 데이터 처리와 캐싱 시스템을 직접 구현해보고 있습니다. 이러한 모든 학습 과정은 기술 블로그에 꾸준히 기록하며 성장의 흔적으로 남기고 있습니다.
